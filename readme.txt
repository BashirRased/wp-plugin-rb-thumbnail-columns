Adds a thumbnail (featured image) column to your WordPress admin post list screen.

== Description ==
RB Thumbnail Columns displays a **Thumbnail Image** column in the WordPress admin for Posts, Pages, and WooCommerce Products.  
This lets you quickly see which content has a featured image.

Lightweight, fast, and uses native WordPress functions only.

### Features
* Adds a thumbnail column to admin post lists
* Supports Posts, Pages, and WooCommerce Products
* Shows thumbnail preview (100×100)
* Sortable thumbnail column
* Filter posts:
  * Has thumbnail
  * No thumbnail
* No configuration needed

== Installation ==

### From your WordPress dashboard:

1. Go to **Plugins → Add New**
2. Search for **"RB Thumbnail Columns"**
3. Click **Install Now** and then **Activate**

That's it — the thumbnail column will automatically appear in your post list screen.

== Frequently Asked Questions ==

= 1. Does this plugin require any other plugin? =
No. It works independently and uses WordPress core functions.
