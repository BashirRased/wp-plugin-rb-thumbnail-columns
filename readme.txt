RB Thumbnail Columns plugin use for your posts visit count.

== Description ==
RB Thumbnail Columns is show your post thumbnail column.

== Installation ==

### From your WordPress dashboard

= Visit =
Plugins > Add New

= Search =
for "RB Thumbnail Columns"

= Install and Activate =
"RB Thumbnail Columns" from your Plugins page

= Go to =
Appearance -> Widgets -> Widgets List: Widgets Category -> RB Thumbnail Columns

= Active Widget =
Actiated your widget and save.
